# a-vue-demo

> A Vue.js project
> 使用了v-router v-resource element-ui
> 豆瓣电影和CNode社区的api

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
